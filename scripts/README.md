# 脚本工具目录

本目录包含开发和调试用的辅助脚本，**不应在生产环境中使用**。

## 📁 文件说明

### test_config.py
**用途**: 测试配置加载是否正常

**使用方法**:
```bash
python scripts/test_config.py
```

**输出**: 显示各项配置的状态（已配置/未配置），但不会输出完整密钥。

---

### debug_llm.py
**用途**: 调试 LLM 提供商加载

**使用方法**:
```bash
python scripts/debug_llm.py
```

**输出**: 显示 LLM 配置状态和已加载的提供商列表。

---

### setup_git.ps1
**用途**: Windows 环境下初始化 Git 仓库（用于部署）

**使用方法**:
```powershell
.\scripts\setup_git.ps1
```

**功能**: 
- 检查 Git 是否安装
- 初始化仓库
- 添加远程仓库
- 首次提交

---

## ⚠️ 安全提示

- 这些脚本仅用于本地开发和调试
- 不要在 CI/CD 或生产环境中运行
- 不要将脚本输出分享给他人（可能包含敏感信息前缀）
